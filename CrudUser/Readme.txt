Routes
GET 	: '/users' 			: mengembalikan semua list user
GET 	: '/users/:id'			: mengembalikan user berdasarkan id yang diberikan
POST	: '/users'			: menambahkan user kedalam database
PUT	: '/users/:id'			: mengupdate data user dengan id yang diberikan
DELETE	: '/users/:id'			: menghapus data user dengan id yang diberikan
